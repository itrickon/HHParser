# HHParser v2.0 - Оптимизированная версия

<a id='readme-top'></a>

Desktop-приложение для анализа данных и сбора информации с HH.ru с модульной архитектурой и улучшенной защитой от детекции.

---

## Что нового в версии 2.0?

### Ключевые улучшения:

- Модульная архитектура - код разбит на логические модули для удобства поддержки
- Улучшенная антидетект защита - 12+ скриптов для скрытия автоматизации
- Полная типизация - все функции имеют type hints
- Unit-тесты - покрытие ключевых модулей тестами
- Файловое логирование - логи сохраняются в файлы с ротацией
- Централизованная конфигурация - все настройки в одном месте
- Валидация данных - проверка URL, файлов, телефонов
- Красивое Excel форматирование - автоматическое форматирование результатов

---

## Возможности

### Режимы работы:

- Парсер телефонов - парсинг контактной информации из объявлений по Excel файлу с URL
- Поиск по URL - сбор вакансий со страницы поиска HH.ru без входа в аккаунт

### Функционал:

- Автоматическое сохранение результатов в Excel
- Поддержка светлой и тёмной темы
- Пагинация: сбор вакансий с нескольких страниц поиска
- Фильтрация рекламных ссылок (adsrv.hh.ru)
- Stealth режим для обхода антибот систем
- Рандомизация fingerprint браузера
- Поддержка прокси
- Human-like поведение (случайные задержки, скроллинг)

## Установка

### Требования:

- **Python**: версия 3.13+ или выше
- **Google Chrome**: браузер для автоматизации
- **Подключение к интернету**
- **Учетная запись HH.ru** (для режима "Парсер телефонов")

### Способ 1: Быстрая установка

```bash
# Клонируем репозиторий
git clone https://github.com/itrickon/HHParser.git
cd HHParser

# Устанавливаем зависимости
pip install -r requirements.txt

# Устанавливаем браузер Playwright
playwright install chromium
```

### Способ 2: Для разработки

```bash
# Устанавливаем все зависимости включая dev
pip install -r requirements-dev.txt

# Запускаем тесты
pytest

# Проверяем код стиль
black src/ tests/
flake8 src/ tests/
```

## Использование

### Режим 1: Парсер телефонов

#### Через GUI:

```bash
python gui.py
```

1. Выберите переключатель **"Парсер телефонов"**
2. Нажмите кнопку **"Excel файл"** и выберите файл с ссылками на вакансии
3. Установите нужное количество фирм для обработки
4. Нажмите кнопку **"Начать парсинг"**
5. Дождитесь открытия браузера Chrome
6. Если требуется авторизация, выполните вход вручную
7. После успешного входа нажмите кнопку **"Вход выполнен"**
8. После завершения нажмите **"Экспорт"** → **"Экспорт готового файла"**

#### Через код:

```python
import asyncio
from src.phone_parser import HHPhoneParser

async def main():
    parser = HHPhoneParser(
        input_file="vacancies.xlsx",
        max_num_firm=20,
        gui_works=False,
    )
    await parser.parse_main()

if __name__ == "__main__":
    asyncio.run(main())
```

### Режим 2: Поиск по URL

#### Через код:

```python
import asyncio
from src.url_collector import HHVacancyCollector

async def main():
    collector = HHVacancyCollector(
        search_url="https://hh.ru/search/vacancy?area=1&text=python",
        max_vacancies=50,
    )
    await collector.parse_main()

if __name__ == "__main__":
    asyncio.run(main())
```

## Конфигурация

Все настройки находятся в файле `config.py`:

### Основные настройки:

```python
# Количество вакансий
MAX_VACANCIES = 20

# Таймауты
NAV_TIMEOUT = 30000  # 30 секунд

# Антидетект
CONCURRENCY = 3  # Одновременных вкладок
BETWEEN_ACTIONS_PAUSE = (0.10, 0.30)  # Паузы между действиями

# Прокси
PROXY_ENABLED = False
PROXY_SERVER = "http://user:pass@proxy:port"

# Логирование
LOG_LEVEL = "INFO"
LOG_FILE = "logs/hh_parser.log"
```

### Включение прокси:

```python
# В config.py
PROXY_ENABLED = True
PROXY_SERVER = "http://myproxy:8080"
PROXY_USERNAME = "user"
PROXY_PASSWORD = "pass"
```

## Логирование

Логи сохраняются в `logs/hh_parser.log` с ротацией (5 файлов по 10MB).

### Уровни логирования:

- `DEBUG` - детальная информация
- `INFO` - общие события
- `WARNING` - предупреждения
- `ERROR` - ошибки
- `CRITICAL` - критические ошибки

## Устранение неполадок

### Парсер не запускается:

```bash
# Проверьте Python
python --version  # Должна быть 3.13+

# Проверьте зависимости
pip install -r requirements.txt

# Установите браузер
playwright install chromium
```

### Парсинг не работает:

- Проверьте подключение к интернету
- Проверьте формат входного файла
- Убедитесь что ссылки в формате: `https://hh.ru/vacancy/87654321`

### Ошибка авторизации:

- Нажмите "Вход выполнен" только после успешного входа
- Очистите кэш браузера
- Запустите парсинг заново
