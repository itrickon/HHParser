# HHParser
<a id='readme-top'></a>

Desktop-приложение для анализа данных и сбора информации.

## Безопасность и соблюдение правил

#### **Данный инструмент предназначен для сбора открытой информации в образовательных и исследовательских целях.** 

### Рекомендуется:  
- Ознакомиться с условиями использования!

- Использовать инструмент ответственно и в соответствии с законодательством!

- Не использовать для спама или нарушения правил площадки!

### Возможности:  
Основной режим работы:

- Парсинг контактной информации из объявлений

- Автоматическое сохранение результатов в Excel

### Программное обеспечение:  

- Python: версия 3.13+ или выше

- Google Chrome: браузер для автоматизации

- Подключение к интернету

- Учетная запись

### Установка  

Установите Python:

- Запустите python_installer_1.bat или скачайте Python с официального сайта ([ссылка на сайт](https://www.python.org/))

- Отметьте галочкой "**Use admin privileges when installing py.exe**"
- Отметьте галочкой "**Add python.exe to PATH**"
- Нажмите на кнопку "**Install Now**"
- После загрузки появится "**Setup was successful**", нажмите на кнопку "**Close**"

### Способ 1:

- Запустите **hhparser_installer_2.bat**

- Дождитесь установки зависимостей и сборки EXE-файла

- На рабочем столе появится ярлык **HHParser**

- Запустите **HHParser**

- Используйте графический интерфейс как описано в **способе 2** (без запуска gui.py)

### Способ 2:

- Запустите **gui.py**

- Нажмите кнопку **"Excel файл"** и выберите файл с ссылками на вакансии

- Установите нужное количество фирм для обработки в поле **"Количество фирм"**

- Выберите тему интерфейса: "Светлая тема" или "Темная тема"

- Нажмите кнопку **"Начать парсинг"**

- Дождитесь открытия браузера Chrome

- Если требуется авторизация, выполните вход вручную

- После успешного входа нажмите кнопку **"Вход выполнен"** в интерфейсе программы

- Программа продолжит автоматический парсинг выбранных объявлений

- После завершения парсинга нажмите **"Экспорт"** → **"Экспорт готового файла"** и выберите папку для сохранения

### Способ 3:

- Откройте **cmd/terminal** и установите зависимости:  
`pip install sv-ttk playwright openpyxl pandas`

- Установите браузер Playwright:  
`playwright install chromium`

- Запустите **hh_phone_search.py** и в функции main укажите:  
`input_file="ваш_файл.xlsx"` - файл со ссылками на вакансии  
`max_num_firm=20` - количество вакансий для обработки  
`gui_works=False` - режим без графического интерфейса  

### Горячие клавиши
Для быстрой работы используйте сочетания клавиш:

|Сочетание|	Действие|
|---|---|
|Ctrl + O|	Открыть Excel файл|
|Ctrl + L|	Очистить лог|
|Ctrl + Q|	Выйти из программы|
|Ctrl + S|	Остановить парсинг|
|Ctrl + G|	Вход выполнен|
|Ctrl + K|	Горячие клавиши|
|F1|	Руководство пользователя|


### Устранение неполадок  

Если парсер не запускается:  

- Проверьте установку Python:

- `python --version`  
Должна отобразиться версия 3.13 или выше

- Если парсинг не работает:  
Проверьте подключение к интернету

- Проверьте формат входного файла:

- Файл должен содержать ссылки на вакансии

- Ссылки должны быть в формате: https://hh.ru/vacancy/12345678

- Проверьте авторизацию

- Нажмите "Вход выполнен" только после успешного входа

- Очистите кэш браузера:  
Закройте все окна Chrome  
Запустите парсинг заново
