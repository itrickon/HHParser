# Changelog

Все заметные изменения проекта HHParser

## [2.0.0] - 2026-04-14

### 🎉 Major Release - Полная оптимизация проекта

#### ✨ Добавлено:

**Архитектура:**
- 🏗 Модульная архитектура с разделением ответственности
- 📦 8 новых модулей в директории `src/`
- 🔧 Централизованная конфигурация (`config.py`)
- 📝 Система логирования с ротацией файлов (`logger.py`)

**Безопасность и антидетект:**
- 🛡️ 12+ stealth скриптов для скрытия автоматизации (`stealth.py`)
- 🎲 Рандомизация User-Agent, viewport, platform, timezone
- 🔌 Поддержка прокси серверов
- 🔍 Валидация всех входных данных (`validator.py`)

**Качество кода:**
- ✅ Полная типизация всех функций (type hints)
- 🧪 Unit-тесты для ключевых модулей
- 📊 Тесты валидации, Excel работы и stealth функций
- 🎨 Автоматическое форматирование Excel файлов

**Документация:**
- 📖 Полностью переписанный README.md
- 🤝 Руководство по контрибьюции (CONTRIBUTING.md)
- 📋 Changelog для отслеживания изменений
- 📝 Inline документация в коде

#### 🔧 Изменено:

**Рефакторинг:**
- ♻️ `hh_phone_search.py` → `src/phone_parser.py` (оптимизирован)
- ♻️ `hh_url_collector.py` → `src/url_collector.py` (оптимизирован)
- ♻️ Общая логика вынесена в `src/base_parser.py`
- ♻️ Работа с браузером в `src/browser_manager.py`
- ♻️ Извлечение данных в `src/vacancy_extractor.py`
- ♻️ Excel операции в `src/excel_manager.py`

**Улучшения:**
- ⚡ Улучшенная обработка ошибок
- 🎯 Валидация URL перед парсингом
- 💾 Сохранение данных каждые 3 записи (было в конце)
- 📈 Логирование в файл и GUI одновременно

#### 🗑 Удалено:

- Дублирование кода между модулями
- Хардкод путей и настроек
- Устаревшие комментарии

#### 📦 Зависимости:

**Добавлены:**
- `pytest>=7.0.0` - тестирование
- `pytest-cov>=4.0.0` - покрытие кода
- `pytest-asyncio>=0.21.0` - async тесты
- `black>=23.0.0` - форматирование
- `flake8>=6.0.0` - линтинг
- `mypy>=1.0.0` - проверка типов
